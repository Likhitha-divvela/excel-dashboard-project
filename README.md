# Restaurant Sales Dashboard

## Project Overview
This Excel dashboard analyzes restaurant sales data and provides insights into sales performance, customer ratings, and city-wise trends.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- VLOOPUP's

## Key Insights
- Total Sales
- Top 5 Restaurants by Sales
- City-wise Performance
- Restaurant Ratings Analysis
